# simple-calculator

This is a simple calculator program written in C++.

## Features

- Addition
- Subtraction
- Multiplication
- Division

## Usage

1. Clone the repository
2. Compile the code using a C++ compiler
3. Run the executable

### Example

```sh
$ g++ -o calculator calculator.cpp
$ ./calculator
Enter first number: 10
Enter second number: 5
Enter operation (+, -, *, /): +
Result: 15
